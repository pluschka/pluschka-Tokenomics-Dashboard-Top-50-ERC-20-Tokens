Glossary
Airdrops - Free token distribution to users for marketing or community growth.
Initial Coin Offering (ICO) - Token sale directly to investors to raise project funds.
Initial Exchange Offering (IEO) - Token sale conducted via a centralized exchange platform.
Security Token Offering (STO) - Regulated token offering representing security-like assets or rights.
Initial DEX Offering (IDO) - Token launch through decentralized exchanges and liquidity pools.
Liquidity Bootstrapping Pool (LBP) - Dynamic pricing pools designed to start fair token distribution.
Fair Launch - No pre-mining; tokens distributed equally through mining or staking.
Direct Sale - Tokens sold directly to selected users or private investors.
Community Incentives - Rewards to community members for participation and engagement.
Bounty Programs - Token rewards for users completing specific promotional tasks.
Other - If none of the above applies.
